# Commande /verify-db

Vérifie l'intégrité des données avec DuckDB.

## Étapes

1. Lancer une vérification DuckDB :
   ```bash
   python scripts/ingest_halo_data.py --action verify
   ```

2. Si des erreurs sont détectées, les documenter dans `.ai/thought_log.md`

3. Proposer des corrections si nécessaire
